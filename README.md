# Welcome to my `Discord.js` template

### Setup

run `npm install`
create `.env` and add the following:
- `BOT_TOKEN`
- `CLIENT_ID`
- `MONGO_URI` *optional remove the connect function at <a href="https://github.com/gamerlord295/discordjs-template/blob/main/server.js">server.js</a>*


to run server `npm run dev` *uses nodemon*
to publish commands run `npm run commands`

### This template is route based
- <a href="https://github.com/gamerlord295/discordjs-template/blob/main/src/commands">src/commands</a> to add commands
- <a href="https://github.com/gamerlord295/discordjs-template/blob/main/src/events">src/events</a> to add events create new file with the name of the event such as: `guildMemberJoin` and etc.
